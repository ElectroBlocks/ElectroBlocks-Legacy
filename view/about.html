About ElectroBlock

A big thanks to BlockDruino for giving the start code for this project. Also a big thank you to google for creating the Blockly meta language and javascript library. I would also like to give a huge thank you and shout out to Arduino for making process or the awesome Arduino IDE. If you are interested in looking at the code it's here.

Features

  • Block Programming
  • Debugging
  • USB Communication (both ways)
  • Automatic Updates & Bug Fixes
  • Arduino Blocks For:
    • Input & Output
    • Neo Pixels
    • RFID
    • Led Matrix
    • LCD Displays
    • Bluetooth
    • Servos
    • IR Remote
    • And More :)

List of Node Packages used

  • Electron
  • AVR Girl
  • RxJS
  • Serial Port
  • axios
  • Electron Prompt
  • Electron Builder
